===== USAGE INSTRUCTIONS =====

1. Go to Google Ads > Audience Manager > Custom Segments
2. Create New Custom Segment
3. Select "People who searched for any of these terms on Google"
4. Copy ALL keywords from "KEYWORDS TO ENTER" section above
5. Paste into the keyword field (one per line)
6. Select keyword setting: "Audience" (recommended)
7. Name the segment: "Property Investment Indonesia - High Intent"
8. Save

9. Create SECOND custom segment for negative keywords:
   - Create exclusion list
   - Add all keywords from "NEGATIVE KEYWORDS" section
   - Apply to all campaigns

===== MONITORING & OPTIMIZATION =====

After 2 weeks, review Search Terms Report:
□ Check which keywords are triggering ads
□ Add irrelevant terms to negative keyword list
□ Identify new relevant keywords to add
□ Create separate segments for high-performers

Success Indicators:
✓ Click-through rate > 2%
✓ Time on site > 2 minutes
✓ Pages per session > 3
✓ Bounce rate < 50%

If performance is poor:
- Try switching to "Content" keyword setting
- Create smaller, more specific segments
- Increase bid adjustments for best performers
